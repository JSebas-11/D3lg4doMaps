# v2.0.0 Breaking Changes

:::danger Breaking Change
This version introduces a **global namespace refactor** affecting all SDK packages:
Core, Places, and Routes.
:::

## Namespace Refactor

Namespaces were **simplified and standardized** across all packages.

### Changes

- Renamed root namespace:
  - `D3lg4doMaps.*` → `DelgadoMaps.*`

- Removed redundant `.Public` segment:
  - `D3lg4doMaps.Core.Public.*` → `DelgadoMaps.Core.*`
  - `D3lg4doMaps.Places.Public.*` → `DelgadoMaps.Places.*`
  - `D3lg4doMaps.Routes.Public.*` → `DelgadoMaps.Routes.*`

---

### Examples

#### Before
```csharp
using D3lg4doMaps.Core.Public.Abstractions;
using D3lg4doMaps.Places.Public;
using D3lg4doMaps.Routes.Public;
```

#### After
```csharp
using DelgadoMaps.Core.Abstractions;
using DelgadoMaps.Places;
using DelgadoMaps.Routes;
```

---

### Why this change?

- **Cleaner and shorter** namespaces
- Removes unnecessary layering (`Public`)
- Improves **developer experience and readability**
